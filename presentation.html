[ PROBLEM STATEMENT ]

High-Stakes Environments Demand More

Air traffic control is a regulated, zero-margin-for-error domain. Most AI agents today operate alone, locked inside one environment or a single workflow.

  • Single-agent chatbots cannot handle multi-faceted complex situations.
  • Opaque decision-making violates regulatory compliance needs.
  • Fully autonomous execution is highly dangerous without human oversight.
[ ARCHITECTURE ]

AI-Assisted, Human-Decided

We built a system where specialised AI agents collaborate through Band to help human controllers detect conflicts, analyse weather, and coordinate emergencies.

The Golden Rule: Agents detect, review, and recommend — but the human controller holds the only authority to execute.

Core Pillars

  • Cross-Framework Collaboration
  • Adversarial Review Loop
  • Human-on-the-Loop Architecture
  • Regulator-Ready Audit Trails

[ PRIORITY OVERRIDE ]

The Emergency Veto (7700)

For an emergency squawk, the loop expands dynamically to recruit Ground Ops for runway info.

Crucially, the Emergency Response agent holds veto power: while an emergency is active, lower-priority conflict and weather dispatches are deferred per real-world ATC priority rules.

[ COMPLIANCE ]

Regulator-Ready Audit Trails

Every agent action is logged for regulatory compliance. Band events (`thought`, `tool_call`, `tool_result`) flow seamlessly into the audit timeline.

Generate a one-click JSON incident report with the full agent reasoning trail and controller decisions.

Adversarial Review

An independent Safety Reviewer cross-examines every advisory against ICAO minima before it reaches the human gate. The safety agent can OVERRIDE, REJECT, or APPROVE peer outputs.

[ AI/ML API INTEGRATION ]

Best Model for the Job

We target the AI/ML API prize by using principled, per-agent model choices rather than forcing a single model everywhere.

  • DeepSeek-V4-Pro: Deep reasoning + reliable JSON for CPA math (Conflict & Weather).
  • Zhipu GLM-5.1: Deterministic parsing at temp=0 for Safety Review & Emergency class.
  • Moonshot Kimi: Long-context instruction following for Coordinator dispatch.
  • DeepSeek-V4-Flash: Fast, cheap tool-calls for Ground Ops lookups.
[ BUSINESS VALUE ]

Beyond Aviation

While designed for ATC, this architecture solves the core problem of any high-stakes enterprise workflow.

Financial services (approvals), legal review (contracts), and healthcare (coordination) all require the same primitives:

  • Specialised agents with distinct knowledge.
  • Adversarial review + Human-in-the-loop gates.
  • Cryptographically verifiable audit logs.
